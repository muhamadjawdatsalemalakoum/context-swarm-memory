The thesis

More memory should mean more clarity, not more noise.

CSM treats memory as bounded, inspectable shards. Queries branch from immutable snapshots, gather cited evidence, synthesize a compact packet, and discard the temporary work. Durable memory changes only through the Committer.

01

Shard the past

Long histories stay organized into bounded witnesses with summaries, tags, and immutable snapshots.

02

Recall with sources

Every useful answer carries shard, snapshot, and event IDs so claims can be checked.

03

Write deliberately

Read paths never mutate durable memory. New facts go through explicit commit decisions.

Architecture

A branch-and-discard read path.

The manager routes a query, probes candidate shards, recalls from useful snapshots, and synthesizes a MemoryPacket. The query run is temporary; snapshots and the chronicle remain untouched.

  • Router is keyword and tag based; starved packets get a local embedding recall floor, with no LLM indexing cost.
  • Probe and recall calls stay behind the LlmProvider seam.
  • Committer is the only durable write entry point.

Safety invariants

Designed for memory you can audit.

CSM avoids autonomous writes and tool-using shards. Reads produce evidence; commits produce durable state. The storage layer refuses snapshot overwrites.

Querying memory must not mutate memory

Mutation-safety tests hash durable files before and after ask runs.

All writes go through Committer

appendEventAndSnapshot and applyCommitDecision are the durable write gates.

Snapshots are immutable

Shard snapshots are versioned and the storage layer refuses overwrite attempts.

Run it

Clone, test, and ask memory locally.

The default provider is deterministic and runs without API keys. Real providers stay behind the same LlmProvider interface. 

npm install
npm test

npm run csm -- init
npm run csm -- shard create --name "Project X" --tags x,architecture
npm run csm -- remember --shard <shardId> --text "Decision: ..." --tags decision
npm run csm -- ask "What did we decide about X?"
